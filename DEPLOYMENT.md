## 🚀 部署方案选择

### 方案 1：部署 H5 版本到服务器（推荐）

**适用场景**：
- 需要在浏览器中访问
- 快速测试和使用
- 不依赖微信生态

**部署选项**：
- ✅ Vercel（免费，自动部署）
- ✅ Netlify（免费，自动部署）
- ✅ 自己的 Nginx/Apache 服务器
- ✅ 阿里云/腾讯云静态网站托管

**优点**：
- 部署简单，几分钟完成
- 免费/低成本
- 不依赖沙箱
- 支持 HTTPS
- 自动 CDN 加速

---

### 方案 2：发布为微信小程序

**适用场景**：
- 需要在微信生态中使用
- 给微信用户使用
- 需要小程序生态功能（分享、支付等）

**部署步骤**：
1. 申请微信小程序账号
2. 下载微信开发者工具
3. 导入 `dist/` 目录
4. 配置小程序信息
5. 上传并提交审核
6. 发布上线

**优点**：
- 微信生态，用户体验好
- 支持微信分享、支付等功能
- 官方分发渠道

**缺点**：
- 需要审核流程
- 需要小程序账号
- 受微信平台限制

---

### 方案 3：部署到自己的服务器

**适用场景**：
- 有自己的服务器
- 需要完全控制
- 企业级应用

**部署内容**：
- 前端：`dist-web/` 部署到 Nginx
- 后端：`server/dist/` 部署到 PM2/Docker

**优点**：
- 完全控制
- 可以自定义配置
- 数据安全

**缺点**：
- 需要服务器运维经验
- 需要配置域名、SSL 证书
- 成本较高

---

## 📋 部署前准备

### 1. 环境变量配置

创建 `.env.production` 文件：

```env
# API 地址（部署后需要修改为实际的后端地址）
PROJECT_DOMAIN=https://your-domain.com
```

### 2. 后端 API 配置

后端服务需要配置：
- 数据库连接（Supabase）
- 文件存储（Supabase Storage）
- 端口（默认 3000）

### 3. 域名和 SSL

- 准备域名（如 `kucuntong.com`）
- 配置 DNS 解析
- 申请 SSL 证书（Let's Encrypt 免费）

## 🎯 推荐部署方案

**如果你想要快速上线，我推荐：**

### 选项 A：部署 H5 到 Vercel（最简单，5分钟搞定）

1. 安装 Vercel CLI：`npm i -g vercel`
2. 运行部署命令：`vercel --prod`
3. 按提示完成配置
4. 获得一个免费域名（如 `kucuntong.vercel.app`）

**优点**：
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 全球 CDN
- ✅ 自动部署
- ✅ 不依赖沙箱

1. 申请微信小程序账号（mp.weixin.qq.com）

3. 导入 `dist/` 目录
4. 填写 AppID 和配置信息
5. 上传代码
6. 提交审核
7. 发布上线
