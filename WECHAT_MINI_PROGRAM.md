# 微信小程序部署指南

## 📱 微信小程序部署完整流程

### 第一阶段：准备工作（30分钟）

#### 1. 注册微信小程序账号

**步骤**：
1. 访问：https://mp.weixin.qq.com/
2. 点击"立即注册"
3. 选择"小程序"
4. 填写账号信息（邮箱、密码）
5. 邮箱验证
6. 选择主体类型：
   - **个人**：适合个人开发者，功能有限制
   - **企业**：适合企业，功能完整
7. 完善主体信息（营业执照等）
8. 填写小程序信息（名称、简介、头像）
9. 激活账号

**重要**：
- 保存 AppID（在小程序管理后台可以看到）
- 个人主体的小程序无法开通支付功能
- 企业主体需要营业执照

---

#### 2. 下载微信开发者工具

**下载地址**：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

**支持平台**：
- Windows
- macOS
- Linux（部分功能）

**安装步骤**：
1. 下载对应平台的安装包
2. 安装微信开发者工具
3. 登录微信账号（与小程序账号一致）

---

#### 3. 配置小程序基本信息

在微信公众平台（mp.weixin.qq.com）配置：

**必填信息**：
- 小程序名称
- 小程序简介
- 小程序头像（1:1 比例，2MB 以下）
- 服务类目（选择"工具"或"其他"）

**服务器域名配置**（开发阶段可以先跳过）：
1. 进入"开发" → "开发管理" → "开发设置"
2. 配置服务器域名：
   - request 合法域名：你的后端 API 地址
   - uploadFile 合法域名：你的后端 API 地址
   - downloadFile 合法域名：你的后端 API 地址

---

### 第二阶段：导入项目（10分钟）

#### 1. 导入代码

**打开微信开发者工具**：

1. 点击"+"号创建新项目
2. 填写项目信息：
   - **项目名称**：库存通（或自定义）
   - **目录**：选择 `dist/` 文件夹
   - **AppID**：填写你的小程序 AppID
   - **开发模式**：小程序
   - **后端服务**：不使用云服务
3. 点击"创建"

#### 2. 验证项目

项目导入后应该看到：
- ✅ 首页显示"活体昆虫库存管理"
- ✅ 可以看到昆虫列表
- ✅ 按钮和功能正常

**如果出现错误**：
- 检查 `dist/` 文件夹是否完整
- 确认 AppID 是否正确
- 查看"调试器"控制台的错误信息

---

### 第三阶段：配置项目（15分钟）

#### 1. 修改 app.json 配置

编辑 `dist/app.json` 文件：

```json
{
  "pages": [
    "pages/index/index",
    "pages/detail/index",
    "pages/statistics/index"
  ],
  "window": {
    "backgroundTextStyle": "light",
    "navigationBarBackgroundColor": "#fff",
    "navigationBarTitleText": "库存通",
    "navigationBarTextStyle": "black"
  },
  "style": "v2",
  "sitemapLocation": "sitemap.json"
}
```

#### 2. 配置服务器域名

在微信公众平台配置：
- 进入"开发" → "开发管理" → "开发设置"
- 添加服务器域名：
  ```
  https://your-api-domain.com
  ```

#### 3. 配置 request 合法域名

由于小程序安全限制，必须配置合法域名：

1. 在小程序管理后台：
   - 开发 → 开发管理 → 开发设置 → 服务器域名
   - 添加你的后端 API 地址（必须是 HTTPS）

2. **开发阶段**：
   - 可以在开发者工具中勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"
   - 位置：右上角 → 详情 → 本地设置

### 第五阶段：上传代码（5分钟）

#### 1. 上传步骤

在微信开发者工具中：

1. 点击右上角"上传"按钮
2. 填写版本号（如：1.0.0）
3. 填写项目备注（如：首次发布）
4. 点击"上传"

上传成功后会显示提示信息。

---

### 第六阶段：提交审核（1-3天）

#### 1. 登录微信公众平台

访问：https://mp.weixin.qq.com/

#### 2. 提交审核

1. 进入"版本管理" → "开发版本"
2. 找到你刚才上传的版本
3. 点击"提交审核"
4. 填写审核信息：
   - 测试账号（提供测试账号信息）
   - 功能页面截图（展示主要功能）
   - 隐私指引（如有涉及用户信息）

#### 3. 等待审核

- **个人小程序**：1-3天
- **企业小程序**：1-7天

审核期间：
- 可以查看审核状态
- 如果被驳回，根据反馈修改后重新提交

---

### 第七阶段：发布上线（5分钟）

#### 1. 发布小程序

审核通过后：

1. 进入"版本管理" → "审核版本"
2. 找到审核通过的版本
3. 点击"发布"

#### 2. 验证发布

发布后：
- 在微信中搜索小程序名称
- 扫码打开小程序
- 测试所有功能是否正常

## ⚠️ 重要注意事项

### 1. 服务器域名要求

- **必须是 HTTPS**
- **域名需要备案**（如果服务器在国内）
- **证书有效**（不能过期）

### 2. 个人小程序限制

个人主体的小程序有以下限制：
- ❌ 无法开通支付功能
- ❌ 无法获取用户手机号
- ❌ 无法分享到朋友圈（只能分享给好友）
- ❌ 部分类目受限

### 3. 审核要点

微信审核关注：
- 功能完整性（不能是空壳）
- 用户隐私（如果涉及个人信息）
- 内容合规（不违反法律法规）
- 服务稳定（后端服务正常运行）

### 4. 版本管理建议

- 使用语义化版本号（如：1.0.0）
- 每次发布记录版本日志
- 测试充分后再发布

## 🔧 常见问题

### Q1: 小程序无法请求数据？

**原因**：服务器域名未配置或不是 HTTPS

**解决**：
1. 在微信公众平台配置服务器域名
2. 确保后端使用 HTTPS
3. 开发阶段可以勾选"不校验合法域名"

### Q2: 图片上传失败？

**原因**：上传域名未配置或后端服务未配置

**解决**：
1. 配置 uploadFile 合法域名
2. 确保后端图片上传接口正常
3. 检查 Supabase 存储配置

### Q3: 审核被驳回？

**常见原因**：
- 功能不完整
- 违反小程序规范
- 服务不稳定

**解决**：
1. 查看驳回原因
2. 根据反馈修改
3. 重新提交审核

### Q4: 如何配置支付功能？

**前提**：必须是企业主体小程序

**步骤**：
1. 开通微信支付
2. 配置商户信息
3. 接入支付 SDK
4. 测试支付功能
