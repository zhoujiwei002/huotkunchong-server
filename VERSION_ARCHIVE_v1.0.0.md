## ✨ 新功能

### 核心功能
1. ✅ **库存管理** - 查看和管理各门店的昆虫库存，支持按门店筛选
2. ✅ **添加昆虫** - 支持添加昆虫品种，可上传图片，自动创建初始库存记录
3. ✅ **销售记录** - 记录销售信息和实收价格，自动扣减库存
4. ✅ **死亡记录** - 记录昆虫死亡情况，自动扣减库存
5. ✅ **库存统计** - 查看库存汇总和统计数据
6. ✅ **串货功能** - 支持门店间库存转移，自动记录操作日志
7. ✅ **操作记录** - 查看所有操作历史，包括操作员、时间、数量等

### 预设数据
- ✅ **9 个仓库位置**
  - 全部、公司总部、王东团队、袁兴彪团队
  - 郭秀华团队、王希强团队、王成兵团队
  - 周纪良团队、秦文胜团队、刘君团队

- ✅ **6 种昆虫品种**
  - 天门螳螂（150元）
  - 天门甲虫（120元）
  - 晋中甲虫（100元）
  - 绥化甲虫（110元）
  - 本溪甲虫（130元）
  - 天门睫角（200元）

## 🔧 技术特点

### 前端
- **框架**：Taro 4 + React 18
- **样式**：Tailwind CSS 4
- **状态管理**：Zustand
- **图标库**：lucide-react-taro
- **跨端兼容**：微信小程序 + H5

### 后端
- **框架**：NestJS 10
- **数据库**：PostgreSQL (Supabase)
- **文件存储**：Supabase Storage
- **图片处理**：Sharp
- **数据库初始化**：自动创建预设数据

## 🎯 使用场景

3. 统计各昆虫品种的库存状态
