## 🚀 快速开始（5分钟）

### 第 1 步：运行检查脚本

```bash
./check-wechat-deploy.sh
```

这会检查你的部署准备情况，并显示下一步操作。

---

### 第 2 步：准备微信小程序账号

1. **注册小程序账号**
   - 访问：https://mp.weixin.qq.com/
   - 点击"立即注册" → 选择"小程序"
   - 填写信息并激活账号

2. **获取 AppID**
   - 登录小程序管理后台
   - 进入"开发" → "开发管理" → "开发设置"
   - 复制你的 AppID（类似：wx1234567890abcdef）

3. **下载微信开发者工具**
   - 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   - 安装并登录微信账号

---

### 第 3 步：配置项目

**方法 1：使用快速配置脚本**
```bash
./setup-wechat.sh
```

**方法 2：手动配置**

1. **配置 AppID**
   ```bash
   nano dist/project.config.json
   ```
   找到 `"appid": "touristappid"`
   替换为你的真实 AppID（如：`"appid": "wx1234567890abcdef"`）

2. **配置后端 API 地址**
   ```bash
   cp .env.production.example .env.production
   nano .env.production
   ```
   修改 `PROJECT_DOMAIN` 为你的后端地址（如：`https://api.your-domain.com`）

---

### 第 4 步：导入项目

1. 打开微信开发者工具
2. 点击"+"号创建新项目
3. 填写信息：
   - 项目名称：库存通
   - 目录：选择 `dist/` 文件夹
   - AppID：填写你的 AppID
   - 开发模式：小程序
   - 后端服务：不使用云服务
4. 点击"创建"

---

### 第 5 步：测试功能

在微信开发者工具中：

1. **开发阶段调试**
   - 右上角 → 详情 → 本地设置
   - 勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"
   - 这样可以在不配置服务器域名的情况下测试

2. **测试功能清单**
   - ✅ 首页加载正常
   - ✅ 库存列表显示
   - ✅ 添加昆虫
   - ✅ 销售登记
   - ✅ 死亡记录
   - ✅ 统计报表
   - ✅ 图片上传
   - ✅ 用户昵称设置

3. **真机调试**
   - 点击"预览"按钮
   - 扫描二维码
   - 在手机微信中测试

---

### 第 6 步：配置服务器域名（上线前必须）

在微信公众平台配置：

1. 登录：https://mp.weixin.qq.com/
2. 进入"开发" → "开发管理" → "开发设置"
3. 配置服务器域名：
   - **request 合法域名**：`https://your-api-domain.com`
   - **uploadFile 合法域名**：`https://your-api-domain.com`
   - **downloadFile 合法域名**：`https://your-api-domain.com`

**注意**：
- 域名必须是 HTTPS
- 域名需要备案（如果服务器在国内）
- 证书必须有效

---

### 第 7 步：上传发布

1. **上传代码**
   - 在微信开发者工具中
   - 点击右上角"上传"
   - 填写版本号和备注
   - 点击"上传"

2. **提交审核**
   - 登录微信公众平台
   - 进入"版本管理" → "开发版本"
   - 点击"提交审核"
   - 填写审核信息
   - 等待审核（1-3天）

3. **发布上线**
   - 审核通过后
   - 进入"版本管理" → "审核版本"
   - 点击"发布"
   - 小程序上线！

## 💡 重要提示

### 关于 AppID

- **测试阶段**：可以使用 `touristappid`（微信开发者工具提供的测试 ID）
- **发布阶段**：必须使用真实的 AppID
- **获取方式**：在微信小程序管理后台的"开发设置"中查看

### 关于服务器域名

- **开发阶段**：可以勾选"不校验合法域名"进行测试
- **生产阶段**：必须配置合法域名（HTTPS）
- **域名要求**：必须备案、使用有效证书

### 关于审核

- **审核时间**：个人 1-3天，企业 1-7天
- **审核重点**：功能完整性、内容合规、服务稳定
- **被驳回**：根据反馈修改后重新提交

### 常见问题

1. **小程序无法请求数据？**
   - 检查是否配置了服务器域名
   - 开发阶段可以勾选"不校验合法域名"

2. **图片上传失败？**
   - 检查 uploadFile 合法域名是否配置
   - 检查后端图片上传接口是否正常

3. **审核被驳回？**
   - 查看驳回原因
   - 根据反馈修改
   - 重新提交
