## 🚀 如果沙箱断开了怎么办？

### 方案一：快速恢复（推荐）
```bash
cd /workspace/projects
./restore.sh
```

### 方案二：手动恢复
```bash
# 1. 检查当前状态
git status

# 2. 查看最近的提交
git log --oneline -5

# 3. 如果有未提交的改动，先提交
git add .
git commit -m "chore: 恢复未保存的改动"

# 4. 启动开发环境
coze dev
```

### 方案三：完全重新初始化（如果项目丢失）
```bash
# 1. 重新初始化项目
coze init ${COZE_WORKSPACE_PATH} --template taro

# 2. 查看项目结构
ls -la

# 3. 检查 Git 历史
git log --oneline -10

# 4. 恢复到之前的版本（如果需要）
git checkout <commit-hash>

# 5. 启动开发环境
coze dev
```
